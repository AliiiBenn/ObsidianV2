## Enfiler un élément avec une liste

---

Pour enfiler un élément dans une File avec une *liste* il faut ajouter l'élément à la fin de la liste car pour *défiler* on va enlever le premier élément de la *liste*.

### Pseudo code de la méthode

```
fonction enfiler(element):
	ajouter l'element dans la liste à la fin
```
