## Savoir si une File est vide avec une liste chaînée

---


Pour savoir si une File est vide à l'aide d'une LC il fait juste retourner la valeur de la méthode native de LC pour savoir si elle est vide.

### Pseudo code

```
fonction est_vide():
	retourner file.est_vide()
```
