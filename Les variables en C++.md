## Les variables en C++

---

Le cours sur les variables en *C++* ressemble beaucoup au cours sur [[Les déclarations en C++]]. La syntaxe d'une variable en *C++* est : 
```type nom = valeur;```

par exemple :

```cpp
#include <iostream>
using namespace std;

int main() {
    float argent = 12.34f;
    cout << argent << endl;
    return 0;
}
```

Le code ci-dessus va créer une variable
