<h2 style="text-align: center;"> Les types de variables csharp </h2>

****

Il existe 4 types principaux de variables :
- Les nombres entiers "*int*"
- Les nombres à virgule "*float*"
- Les booléens "bool"
- Les chaines de caractères "*string*"
